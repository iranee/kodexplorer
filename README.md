# Readynas KodExplorer Applications 
KodExplorer for Netgear Readynas 是运行在网件ReadyNas机上基于PHP的私有云存储服务。
App内部集成KOD可道云（v4.37, 2018-11-28），支持在线升级。

### 安装程序
* 1， 下载以 **.deb** 结尾的安装程序
* 2， 进入ReadyNAS管理界面，点进 **应用程序**
* 3， 选择上传并安装应用程序
